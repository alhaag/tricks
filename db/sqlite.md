## Instalação

Quase todas as distribuições linux possuim o sqlite instalado por padrão

Para verificar a versão:

```
$ sqlite3 --version
3.7.17 2013-05-20 00:56:22 118a3b35693b134d56ebd780123b7fd6f1497668
```
